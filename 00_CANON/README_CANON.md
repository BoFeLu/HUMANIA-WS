# Canon (NUEVO_3er_PROYECTO_HUMANIA)

Este directorio es el **hogar canónico** de procedimientos, normas, plantillas y snapshots.
Objetivo: continuidad perfecta entre chats, minimización de errores operativos y máxima fluidez.

## Principios
- El chat NO es el repositorio; el chat es un **terminal de trabajo**.
- Todo lo que sea “regla”, “prohibición”, “procedimiento” o “plantilla” vive aquí.
- Cambios pequeños, verificables, y reproducibles.
- Evitar comandos frágiles: preferir heredocs `cat <<'EOF' > file` y bloques completos.

## Estructura
- 00_CANON: constitución y normas canónicas
- 01_WORKFLOW: flujo diario / sesiones
- 02_TEMPLATES: plantillas (ACK, cierre, prompts, etc.)
- 03_TRICKS: técnicas y trucos (copypaste, heredoc, etc.)
- 04_BUNDLES: procedimiento de bundle y empaquetados
- 05_STATUS: status reports y context summary (snapshots)
- 99_ARCHIVE: material antiguo o descartado
